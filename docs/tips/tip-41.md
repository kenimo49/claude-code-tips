# Tip 41: Automation of automation

> 自動化の自動化

**Category**: Automation
**Related**: [Tip 12: Invest in workflow](./tip-12.md), [Tip 37: Personalized software](./tip-37.md)

## Overview

最終的には、自動化の自動化がすべて。より生産的になるだけでなく、プロセスも楽しくなる。

## The Journey

1. **ChatGPT時代**: コピペとコマンド実行を自動化したくて[Kaguya](https://github.com/ykdojo/kaguya)を構築

2. **Claude Code時代**: 幸いにも、このようなツールを自分で作る必要がなくなった

3. **次のステップ**: タイピングも自動化したい → Claude Codeで音声認識アプリを構築

4. **さらに先**: 繰り返しをCLAUDE.mdに記録、スキルを作成、スクリプトで自動化

## The Principle

同じことを繰り返していると気づいたら：
- 2〜3回は問題ない
- 何度も繰り返すなら、自動化を検討

## Examples

- 同じコマンドを何度も実行 → スクリプト化
- 同じ指示を何度も出す → CLAUDE.mdに記録
- 同じワークフロー → スキルやコマンドを作成

## Why This Matters

- 繰り返し作業を減らす
- プロセスを楽しく
- 生産性向上
