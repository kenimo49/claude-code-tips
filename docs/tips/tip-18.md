# Tip 18: Claude Code as a writing assistant

> Claude Codeをライティングアシスタントとして使う

**Category**: Writing
**Related**: [Tip 19: Markdown](./tip-19.md), [Tip 2: Voice input](./tip-02.md)

## Overview

Claude Codeは優れたライティングパートナー。音声入力と組み合わせると特に効果的。

## The Workflow

### 1. First Draft

コンテキストと詳細な指示を与える（音声入力推奨）：

```
> [何を書きたいか、背景、トーンなど詳細に説明]
```

満足いくまで何度か試す。

### 2. Line-by-Line Review

一緒に確認しながら編集：

```
> 一緒に見ていこう。この行はこの理由でいいね。
> この行はあっちに移動した方がいい。
> ここはこういう風に変えて。
```

### 3. Reference Materials

必要に応じて参考資料を確認：

```
> 類似のスタイルガイドを参照して
```

## Recommended Setup

ターミナルを左に、コードエディタを右に配置：

```
[Terminal: Claude Code]  |  [Editor: document.md]
```

## Why This Matters

対話的なやり取りで文章を磨ける。一方的に生成させるのではなく、協力して書く感覚。
