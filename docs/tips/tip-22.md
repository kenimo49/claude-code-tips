# Tip 22: The best way to get better at using Claude Code is by using it

> Claude Codeを使いこなすには、使い続けること

**Category**: Learning
**Related**: [Tip 43: Keep learning](./tip-43.md), [Tip 12: Invest in workflow](./tip-12.md)

## Overview

ワールドクラスのロッククライマーが「どうすれば上手くなる？」と聞かれて「ロッククライミングをすることだ」と答えた。Claude Codeも同じ。

## The Billion Token Rule

10,000時間ルールの代わりに「10億トークンルール」。

- AIを使いこなすには大量のトークンを消費する
- 直感を養うには実践あるのみ
- Opus 4.5は十分パワフルで手頃なので、複数セッションを同時実行可能

## Supplementary Learning

補助的に役立つもの：
- 動画を見る
- 本を読む
- Tipsを学ぶ

しかし、実際に使うことが一番の学習。

## Why This Matters

- トークン使用量を気にしすぎない
- 実験を恐れない
- 失敗から学ぶ
