# Tip 43: Keep learning!

> 学び続ける！

**Category**: Learning
**Related**: [Tip 22: Learning by using](./tip-22.md), [Tip 42: Share knowledge](./tip-42.md)

## Overview

Claude Codeについて学び続けるための効果的な方法。

## Learning Methods

### Ask Claude Code Itself

Claude Codeについて質問があれば、直接聞く。機能、スラッシュコマンド、設定、フック、MCPサーバーなどについて回答できる専用サブエージェントがある。

### Check Release Notes

```
/release-notes
```

現在のバージョンの新機能を確認。最新機能を学ぶ最良の方法。

### Learn from Community

[r/ClaudeAI](https://www.reddit.com/r/ClaudeAI/) subredditで他のユーザーから学ぶ。ワークフローや使い方のヒントが共有されている。

### Follow Ado's Tips
