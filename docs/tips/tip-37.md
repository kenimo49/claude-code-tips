# Tip 37: The era of personalized software is here

> パーソナライズソフトウェアの時代が来た

**Category**: Concepts
**Related**: [Tip 31: Universal interface](./tip-31.md), [Tip 12: Invest in workflow](./tip-12.md)

## Overview

AI、特にClaude Codeにより、個人向け・小規模プロジェクト向けのソフトウェアをより多く作れるようになった。

## Examples

### Custom Tools Built

- カスタム音声認識ツール（毎日使用）
- Claude Codeのカスタマイズ方法
- データ可視化・分析タスク

### Slack MCP Alternative

[korotovsky/slack-mcp-server](https://github.com/korotovsky/slack-mcp-server)はDockerコンテナで動作する設計。自分のDockerコンテナ内で使うのが難しかった。

→ Claude CodeにSlack Node SDKを使ったCLIを直接書かせた。うまく動いた。

## The Opportunity

やりたいことがあれば、Claude Codeに頼める。小さければ1〜2時間で構築可能。

## Why This Matters

- 汎用ツールに依存しなくて良い
- 自分専用のソリューション
- 素早いプロトタイピングと実装
