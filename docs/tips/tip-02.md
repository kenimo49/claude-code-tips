# Tip 2: Talk to Claude Code with your voice

> 音声入力を使ってClaude Codeとより速くコミュニケーションする

**Category**: Input
**Related**: [Tip 10: Cmd+A and Ctrl+A](./tip-10.md), [Tip 38: Input box navigation](./tip-38.md)

## Overview

タイピングより音声入力の方が速くコミュニケーションできる。ローカルの音声認識システムを使うと効率的。

## Recommended Tools (Mac)

- [superwhisper](https://superwhisper.com/)
- [MacWhisper](https://goodsnooze.gumroad.com/l/macwhisper)
- [Super Voice Assistant](https://github.com/ykdojo/super-voice-assistant) - オープンソース

## Why Local Models Work

ローカルモデルでも十分な精度が得られる。誤認識があっても、Claudeは文脈から正しく解釈できる。

例：「ExcelElanishMark」→「exclamation mark」、「advast」→「Advanced」と解釈される。

## Tips

- **周囲に人がいる場合**: イヤホンに向かって小声で話す（Apple EarPodsなど）
- **飛行機内でも使える**: マイクに近づいて話せば、周囲の騒音があっても認識される
- 友人に話しかけるように自然に話す

## Why This Matters

テキストより音声の方が多くの人にとって速い。電話やボイスメッセージと同じ感覚で使える。
