# Tip 5: AI context is like milk; it's best served fresh and condensed!

> コンテキストは新鮮で凝縮されている方が良い

**Category**: Context
**Related**: [Tip 8: Proactive compaction](./tip-08.md), [Tip 23: Clone conversations](./tip-23.md)

## Overview

新しい会話を開始した時、Claude Codeは最高のパフォーマンスを発揮する。過去のコンテキストを処理する必要がないため。

## The Problem

会話が長くなるにつれて：
- コンテキストが増加
- パフォーマンスが低下する傾向

## The Solution

- **新しいトピックには新しい会話**を開始
- パフォーマンスが低下したら新しい会話を開始
- 長い会話は適切なタイミングでリセット

## Why This Matters

- 新鮮なコンテキスト = より良い応答
- 凝縮されたコンテキスト = より効率的な処理
- 不要な履歴は削ぎ落とす
