# Tip 31: Claude Code as the universal interface

> Claude Codeは万能インターフェース

**Category**: Concepts
**Related**: [Tip 27: Research tool](./tip-27.md), [Tip 37: Personalized software](./tip-37.md)

## Overview

CLIは新しいIDEというだけでなく、Claude Codeはコンピュータ・デジタル世界への万能インターフェース。

## Examples

### Media Editing

```
> この動画を短くして
→ ffmpegで処理

> これらの音声ファイルを文字起こしして
→ Whisperで処理
```

### Data Analysis

```
> このCSVデータを可視化して
→ Python/JavaScriptで処理
```

### System Maintenance

```
> ストレージが足りない。アドバイスして
→ 大きなファイルを探索
→ Final Cut Proの巨大ファイルを発見
→ 未使用のDockerイメージを発見
→ 不要なキャッシュを発見
```

### Internet Access

- Reddit / GitHub / MCPを通じて様々な情報にアクセス

## The Return to Text

コンピュータはテキストインターフェースから始まった。今、テキストインターフェースに戻りつつある。

ただし今回は：
- 3つ、4つ、5つのタブを同時に開ける
- セカンドブレイン、サードブレイン...
- 退屈なタスクを委任できる（例：GitHub Actionsログ調査）
